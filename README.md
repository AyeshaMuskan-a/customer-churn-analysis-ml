# 📊 Customer Churn Prediction & Credit Risk Analysis

## 📌 Project Overview

This project focuses on analyzing customer behavior and predicting churn using Machine Learning, along with SQL-based credit risk analysis and an interactive dashboard for business insights.

---

## 🎯 Objectives

* Predict whether a customer will churn or not
* Identify key factors influencing churn
* Analyze customer financial risk using SQL
* Build an interactive dashboard for decision-making

---

## ❗ Problem Statement

Businesses face high customer churn, leading to revenue loss. Identifying high-risk customers and understanding their behavior is essential to improve retention strategies and reduce churn.

---

## 📂 Dataset Description

| Column         | Description                          |
| -------------- | ------------------------------------ |
| customerID     | Unique customer identifier           |
| Tenure         | Number of months customer stayed     |
| MonthlyCharges | Monthly bill amount                  |
| TotalCharges   | Total amount charged                 |
| Churn          | Whether customer left (1) or not (0) |
| AvgCharges     | Average charges per tenure           |

## 🔑 Key Insights

* Customers with high monthly charges are more likely to churn
* Low tenure customers have higher churn probability
* Certain features strongly influence churn prediction
* Financial risk indicators help identify high-risk customers
